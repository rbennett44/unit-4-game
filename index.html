Crystal Collection

You will be given a random number at the beginning of the game.

There are four crystals below. By clicking on a crystal you will add a number to your points. You win the game if you can get the total to match the random number assigned by the computer. You lose if you go over.

The value of each crystal is hidden from view. You will need to remember what each one is worth.

Each time the game resets, another random number is picked and the values of the crystals will change.

Good luck!
